GhostNote Bundle

Contents
  - GhostNote.app (double-click to run)
  - source/ (Python + HTML source)

Run
  - Double-click GhostNote.app
  - Logs: ~/Library/Application Support/GhostNote/ghostnote.log

Notes
  - The app opens http://127.0.0.1:8765/index.html and connects to 127.0.0.1:8766.
  - On macOS, it prefers opening in Google Chrome when installed.
  - If sounddevice fails to load on macOS, install PortAudio with: brew install portaudio
